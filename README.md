# Online converter for markdown files

Online converter that should accept markdown files and convert those files to ebook formats for example pdf, epub or mobi files.

## Tech stack

- Frontend: Next.js, Tailwind CSS, TypeScript
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL

## Functional requirements

- User can create an account
- User can login
- User can upload markdown file
- User can select output format (pdf, epub, mobi)
- User can convert file
- User can download converted file
- User can view his file history
